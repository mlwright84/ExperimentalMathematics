# Experimental Mathematics

This repository contains Mathematica and Python code files to accompany the text _Experimental Mathematics: A Computational Perspective_ by Matthew P. Richey and Matthew L. Wright, published 2025 by MAA Press.
The book is available at [https://bookstore.ams.org/view?ProductCode=TEXT/74](https://bookstore.ams.org/view?ProductCode=TEXT/74).
